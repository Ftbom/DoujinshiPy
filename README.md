# DoujinshiPy

多合一本子库

## 安装

**需要先安装unrar和redis**

```shell
python install.py
```

## 运行

```shell
python app.py
```

## 支持的类型

* 本地文件
  >支持zip、rar和7z
* 云盘文件
  >支持zip
* 来自网站的内容

## 使用

* API参考
  >`http://127.0.0.1:9000/docs`或`http://127.0.0.1:9000/redoc`
* doujinshi管理后台
  >`http://127.0.0.1:9000/admin`
* [tachiyomi插件]()
